About This System

This system provides tools for managing inventory and monthly orders through both a web interface and a Discord bot.

The system allows you to:

  • Monitor current inventory levels
  • View current and past monthly orders
  • Manually control order periods (open/close monthly ordering)
  • Add, modify, and delete orders

Discord Bot
Available Commands
  • !inventory - Show current inventory
  • !current_orders - Show current month's orders
  • !past_orders MM/YYYY - Show past month's orders
  • !order - Place an order for the current month
  • !cancel_order - Cancel your order for the current month
  • !products - List all available products
Admin Commands
  • !open_month MM/YYYY - Open a new order month
  • !toggle_month MM/YYYY - Open/close an order month
  • !update_stock ID QTY - Update inventory
  • !add_product "name" "description" - Add a new product
API Endpoints
GET Endpoints
  • /api/inventory - Get current inventory
  • /api/products - Get all products
  • /api/order_periods - Get all order periods
  • /api/order_periods/current - Get current order period
  • /api/orders?period_id=X - Get orders for a period
POST Endpoints
  • /api/order_periods - Create new order period
  • /api/order_periods/<id>/toggle - Toggle order period
  • /api/orders - Add/update an order
DELETE Endpoints
  • /api/orders/<id> - Delete an order
